Abstract

Our project focuses on autonomous navigation using the Moorebot Scout robot. The robot is accessed through SSH, and Python scripts are run within its ROS1 environment to test movement and navigation behavior. The goal is to better understand how software can be used to control a mobile robot. This project also helps demonstrate the connection between Python, ROS1, and real-world robotics applications.
