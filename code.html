Tech Stack

The project is built on a Python-first stack running on the Scout's onboard compute, with a ROS-style messaging layer for sensors and control.

Getting Started

Clone the repository and install dependencies:

$ git clone https://github.com/your-org/moorebot-scout.git $ cd moorebot-scout $ python -m venv .venv $ source .venv/bin/activate $ pip install -r requirements.txt $ python -m scout.run

Example: Obstacle Avoidance

A simplified loop that reads LiDAR distances and steers the robot away from nearby obstacles.

from scout.hardware import LiDAR, Motors def avoid_obstacles(threshold=0.4): lidar = LiDAR() motors = Motors() while True: scan = lidar.read() front = min(scan.front) if front < threshold: motors.stop() motors.turn(direction="left", speed=0.3) else: motors.forward(speed=0.5)

Example: Camera Object Detection

Runs a lightweight model on the onboard camera feed and publishes detected objects.

import cv2 from scout.vision import Detector detector = Detector(model="yolov8n.pt") cap = cv2.VideoCapture(0) while cap.isOpened(): ok, frame = cap.read() if not ok: break for obj in detector.detect(frame): print(f"{obj.label} @ {obj.confidence:.2f}")
