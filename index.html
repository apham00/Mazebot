Welcome to the Mazebot Scout Project

What is Mazebot Scout?

The Mazebot Scout is a compact, autonomous mobile robot platform used in our robotics research project. Our team is programming it to navigate environments, avoid obstacles, and complete tasks — all without human control.

Project Overview

This project is part of our coursework in the Department of Computer Science at Cleveland State University. It focuses on using the Mazebot Scout platform with Python and ROS1 to test autonomous navigation and robot control. We are exploring topics in autonomous navigation, sensor fusion, and real-time control systems using the Mazebot Scout hardware platform.

Goals & Objectives

Our primary objectives include implementing obstacle detection and avoidance, mapping a known environment, and executing a predefined navigation route autonomously. We are also experimenting with camera-based object recognition as an additional goal.
